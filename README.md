# TranslatePPTText_WithOpenAI
PowerPointに設置しているテキスト図形からテキストを抽出し、指定言語から指定言語へとOpenAIAPIを用いたテキスト変換を行います

# PPT翻訳アプリ

## 概要
本アプリは、PowerPoint（.pptx）ファイル内のテキストを  
OpenAI API を使用して翻訳するデスクトップアプリです。

- ロジック：Python
- GUI：customtkinter
- 出力：翻訳後のPowerPointファイル

---

## 主な機能
- PPTファイル内テキストの自動翻訳
- レイアウトを保持したまま翻訳
- 翻訳言語の指定（例：日本語 → 英語）
- GUI操作のみで完結（コマンド不要）

---

## 動作環境
- Windows 10 / 11
- インターネット接続（OpenAI API通信のため）

※ Mac版 / Linux版は未対応

---

## 使用方法
1. exeファイルを起動します
2. 翻訳したいPPTファイルを選択します
3. 翻訳元・翻訳先言語を指定します
4. 「翻訳開始」ボタンをクリックします
5. 翻訳後のPPTが指定フォルダに出力されます

---

## OpenAI APIキーの設定
本アプリは OpenAI API を使用します。  
事前に APIキーを環境変数として設定してください。

### 設定方法（Windows）
1. Windowsの「環境変数」を開く
2. 新規作成  
   - 変数名：`OPENAI_API_KEY`
   - 値：ご自身のAPIキー
3. PCを再起動

---

## 注意事項
- 翻訳処理はOpenAI APIを使用するため、API利用料金が発生します
- 機密情報を含むPPTを翻訳する場合は、社内ルールを遵守してください
- 翻訳精度は内容により異なります
